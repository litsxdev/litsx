# `@litsx/shiki-languages`

Shared Shiki and TextMate language registrations for LitSX-authored TSX and JSX.

This package is the shared source of truth for:

- LitSX syntax highlighting in docs integrations
- generated TextMate grammars for editor tooling

It exports:

- `litsxTsxLanguage`
- `litsxJsxLanguage`
- `litsxCodeLanguages()`
